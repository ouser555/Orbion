# Orbion space mouse 
  
## Orbion是很棒的設計，不過原版Orbion有幾個坑，需要DIY者自行解決

  1.列印件的STL檔有些地方尺寸不對，需要修改
  
  2.Firmware的Encorder部分有問題，大部分市面上的編碼器會動作不正常
  
  3.需要買的零件孔位會對不到
  
    a.joystick模組，孔位有可能會買到差一點的，可以靠擴孔解決。
  
    b.encorder模組，這個底板孔位和編碼器柄長，底座高度都會不一樣，會很不容易買到需要的。
  
  4.三個金屬按鍵缺點(但是好看)
    
    a.很貴
    
    b.不好買
    
    c.不好按
    
    d.擺放位置不好
